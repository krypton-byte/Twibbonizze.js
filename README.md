# Twibbonizze
```bash
> npm i twibbonizze --save
```
# Example

```nodejs
> const {twisearch, Twibbonizze} = require('twibbonizze')


> twisearch('tadikamesra').then(async (x)=>{
    console.log(await x[0].create('image.jpg'))
})
```
